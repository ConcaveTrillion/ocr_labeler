# Usage Docs

End-user documentation for the OCR Labeler app. For developer setup,
contributions, and architecture, see [`DEVELOPMENT.md`](../../DEVELOPMENT.md)
and [`docs/architecture/`](../architecture/README.md).

## Contents

- [Installing](installing.md) — install the app via `curl | sh`,
  `make install`, or from source.
- [How to Label a Page](how-to-label-a-page.md) — workflow for opening a
  project, reviewing OCR output, editing words and lines, tagging styles,
  and saving labeled pages.
